LLM RAG System & Evaluation

A production-ready Retrieval-Augmented Generation system for Persian finance and tax law documents. Combines hybrid BM25 + vector retrieval over a Chroma index with a streaming FastAPI backend and React frontend. Includes a built-in evaluation framework with regression gates that block index deployments when answer quality drops — ensuring the system stays reliable as documents are updated.

Climate AI · Microsoft

End-to-End ML Pipeline for Global Scale Downscaling

A large-scale machine learning pipeline for statistical downscaling of global climate model outputs to high spatial resolution, developed at Microsoft. The system covers the full ML lifecycle — from raw climate model ingestion and feature engineering to distributed model training, automated validation, and deployment-ready forecast outputs — enabling fine-grained climate projections at continental and global extents.

Open Source · CVPR 2025

Python Packaging & Data Benchmarking

Open-source Python package and benchmark dataset for county-level crop yield forecasting across 70+ California crops (2008–2022). Integrates satellite imagery (Sentinel/MODIS), climate variables, evapotranspiration, and soil layers into structured dataloaders. Published to HuggingFace Hub with evaluation protocols and baseline model results, providing a reproducible benchmark for the agricultural ML community.
